
El proyecto fue desarrollado con el propósito de aplicar los principios fundamentales de Programación Orientada a Objetos mediante una solución teórica y una solución funcional, siendo la solución teórica una landing page educativa sobre POO en Java, mientras que la solución funcional corresponde a un sistema de tienda desarrollado en el mismo Java.

## Estructura General Del Proyecto

El proyecto fue dividido en 2 subproyectos:

## LANDING PAGE

Desarrollado con:

-CSS

-Java Script

-INDEX 

## Sistema Funcional

Aplicacion Java Orientada a Objetos, esta se desarrollo mediante:

-Clases

-Herencia

-Encapsulamiento

-Poliformismo

-Exepciones

## Lenguajes

-INDEX.html

-CSS

-Java Script

## Herramientas

-Github

-Visual Studio Code

## Metodologia

El proyecto fue desarrollado utilizando separación de responsabilidades:

docs-project : documentación

poo-project : contenido educativo

funcional-project : aplicación funcional

## Conclusiones

El proyecto permitio aplicar correctamente los conceptos fundamentales de POO, mediante una aplicacion educativa y una funcional.
