## Asignatura

Programacion Orientada a Objetos

En esta organización encontrará de forma detallada los códigos utilizados, así como assets y otros readme específicos para cada repositorio. El presente proyecto fué divido en dos subproyectos, el primero, siendo un landing page dinámicos con propósitos educativos, yendo desde las bases de POO hasta las prácticas de un programador. En el segundo subproyecto, se puso a prueba nuestros conocimientos sobre POO creando un sistema de tienda eficiente donde se tuvieron en cuenta validaciones, excepciones, herencia, polimorfismo, separación de responsabilidades y obviamente, la creación de objetos.
